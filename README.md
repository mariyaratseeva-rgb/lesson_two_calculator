# 🌱 Калькулятор себестоимости саженца

Веб-приложение для расчёта себестоимости саженцев из партии черенков. Подходит для питомников, частных садоводов и небольших хозяйств.

## Возможности

- Расчёт количества укоренённых саженцев с учётом процента укоренения
- Учёт расходов: черенки, субстрат, горшки, удобрения, электроэнергия, труд, прочее
- Автоматический расчёт себестоимости, цены продажи и прибыли
- Мгновенное обновление результатов при изменении полей

## Быстрый старт

1. Склонируйте репозиторий или скачайте архив
2. Откройте `index.html` в браузере

Локальный сервер (опционально): `python -m http.server 8080` → [http://localhost:8080](http://localhost:8080)

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите проект
2. В настройках репозитория: **Settings → Pages → Build and deployment → GitHub Actions**
3. При push в ветку `main` или `master` сайт опубликуется автоматически

## Формулы расчёта

| Показатель | Формула |
|---|---|
| Укоренённые саженцы | `черенки × процент укоренения / 100` |
| Себестоимость | `общие расходы / укоренённые саженцы` |
| Цена продажи | `себестоимость × (1 + наценка / 100)` |
| Прибыль с саженца | `цена продажи − себестоимость` |
| Прибыль с партии | `прибыль с саженца × укоренённые саженцы` |
